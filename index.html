No. 01

Predictive Maintenance on NASA C-MAPSS

In progress · 2026 · RUL estimation on industrial sensor data
  • Estimating Remaining Useful Life (RUL) of turbofan engines from multivariate sensor time-series, using NASA's C-MAPSS benchmark dataset (FD001 to FD004).
  • Planned pipeline: feature engineering on per-engine sensor sequences, baseline tree models (Random Forest, XGBoost), then sequence models (LSTM, 1D-CNN), evaluated with RMSE and an asymmetric loss that penalises late predictions more than early ones.
  • Target deployment: FastAPI inference service plus a Streamlit dashboard for live RUL prediction on input sensor windows. Live demo hosted on Streamlit Cloud / Hugging Face Spaces.
  • Builds directly on baggage-handling fault-trend work from the MAHB internship: industrial sensor data, failure prediction, ops impact.
